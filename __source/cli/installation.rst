Installation
============

.. code-block:: console

   $ git clone "https://gitlab.irstea.fr/evalhyd/evalhyd-python.git"
   $ cmake -S. -Bbuild
   $ cmake --build build

.. admonition:: Requirements

   A C++ compiler and the following libraries are required to install `evalhyd-cli`:

   .. code-block:: text

      cmake
